Topic 1: JavaScript

Loops

In the FOR loop example, the button will have the user enter a number for a multiple. The loop will go through and print the timestables up to the entered number multiplied by 12.

In the WHILE loop example, the button will prompt the user to enter a number. This number will be used to run the WHILE loop until the user's number exceeds 20, after which it will end.

Conditional Statements

In this example of IF, ELSE, IF, the user will enter their age. The IF, ELSE, IF statement will evaluate the user's eligibility to see a PG-13 movie.

In this example of a SWITCH statement, the user will enter their name. The SWITCH statement will determine if they should be allowed entry to the Batcave, and will do so provided they say the right two names, and have other results for other names.

Variables, Parameters, & Functions

This example uses Variables and Parameters in the Function. Pressing the button will prompt the user for their name, occupation, and spouse's name, then call another function to print a message to the user, using the entered data as parameters.
